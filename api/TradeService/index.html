TradeRequestOutBound​

This item only works when running on the server. Server
</>
TradeService.TradeRequestOutBound(
Sender: Player,
Reciever: Player,
Header: string?,
Body: string?,
Blurred: boolean?,
Button1: string?,
Button2: string?
) → PHeSignal

A handler for sending out trade requests, by default it will use prompt service and prompt the reciever, it will then use the reponse from the reciever to either send back "accept" or "decline".

To add your own TradeRequestOutBound, overwrite the function at the top of your source. You will need to also handle how the client precieves this request, by either using remote events or PromptService:PromptUserAsync

warning

You must return a PHeSignal/BindableEvent within your function

You don't have to worry about if either players are in an active trade because the constructor handles that for you.
